# Товары — Административная панель

Приложение для управления списком товаров с авторизацией, сортировкой, поиском и добавлением новых позиций.

## Стек

- **React 19** + **TypeScript**
- **Vite** — сборка и dev-сервер
- **Zustand** — стейт-менеджер
- **React Router** — маршрутизация
- **Lucide React** — иконки
- **React Hot Toast** — уведомления

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev
```

Приложение будет доступно по адресу: **http://localhost:5173**

## Сборка

```bash
npm run build
npm run preview
```

## Авторизация

API: [dummyjson.com/auth/login](https://dummyjson.com/docs/auth)

Тестовый аккаунт:
- **Логин:** `emilys`
- **Пароль:** `emilyspass`

Чекбокс «Запомнить данные»:
- ✅ Включён — токен сохраняется в `localStorage` (сессия переживает перезагрузку)
- ❌ Выключен — токен в `sessionStorage` (сессия сбрасывается при закрытии вкладки)

## Функциональность

- **Список товаров** — загрузка из [dummyjson.com/products](https://dummyjson.com/docs/products), пагинация, progress bar
- **Сортировка** — по цене и рейтингу (ASC/DESC)
- **Поиск** — через API (`/products/search`)
- **Добавление товара** — модальная форма (название, цена, поставщик, артикул), без сохранения через API
- **Рейтинг < 3.5** — подсвечивается красным
